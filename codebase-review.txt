OVERVIEW:
---------
A single-page portfolio website built with React 19, TypeScript, and Vite.
Uses scroll-based view rendering with Intersection Observer for dynamic 
component loading. Deployed on Vercel.

Tech Stack:
- React 19.2.0 with React Compiler (babel-plugin-react-compiler)
- TypeScript 5.9.3
- Vite 7.2.4
- ESLint 9 with flat config
- Vercel Speed Insights
